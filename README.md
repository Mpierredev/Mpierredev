### 🏆 Projetos em Destaque

Aqui estão os projetos que melhor demonstram minha capacidade analítica e técnica:

#### 1. [Performance de Vendas no Varejo (SQL + Tableau)](https://github.com/Mpierredev/analise-desempenho-varejo.git)
* **O que fiz:** Pipeline completo de BI analisando $456k em faturamento. Modelagem de base bruta no PostgreSQL criando uma camada semântica e dashboard executivo.

* **O que fiz:** Diagnóstico de evasão de clientes (Churn) utilizando uma base da IBM. Limpeza de dados via Python e estruturação analítica.
* **Impacto:** Descoberta de gargalos operacionais críticos (ex: 42% de evasão em contratos mensais e alta taxa de cancelamento em fibra óptica), com plano de ação direto para as equipes de Produto e Retenção.
